Project in console

Visual Studio code is one of the many programs you can use to code. It can easily be installed by heading to their website and downloading the exe file which you can then run. It will automatically install VSC to your computer.

The console is a neat part of a website, and can be used to display the outcome of a certain thing like adding or subtracting, showing values, or displaying a message. The console can be accessed by right clicking anywhere on a website, clicking "inspect" and then clicking "console".
